finger-mover踩坑记录

发表于 2017-09-09 | 分类于 frontend

前言

在vue新项目中，尝试使用了finger-mover作为移动端滑动的解决方案，因为是初次在vue中使用finger-mover，所以不可避免的踩了点坑，这里略做记录。

场景描述

使用的是finger-mover的一个插件——simulation-scroll-y，用于纵向的模拟滚动。项目中用了一个loadMore(无限滚动)的功能，加载没有问题，但是对于首次加载的数据条目超过屏幕的宽度的时候，会发现屏幕的滚动出现了问题——无法滑动查看末尾的数据。
